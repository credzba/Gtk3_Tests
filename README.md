# GTK3 Tests

A collection of GTK# 3.0 example applications and experiments for Linux using Mono.

## Projects

### [HelloWorld](./HelloWorld/)
A simple GTK# application demonstrating basic window creation, scrolling, and VSCode debugging setup with Mono.

## Requirements

- Mono 6.8+ with development tools
- GTK# 3.0
- VSCode (optional, for debugging)

## Building

Each project contains its own Makefile. Navigate to the project directory and run:
```bash
make          # Release build
make debug    # Debug build
```

## Debugging with VSCode

Projects are configured for F5 debugging in VSCode using Mono's debugger agent. See individual project READMEs for details.
